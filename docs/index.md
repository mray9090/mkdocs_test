# Infrastructure Team Documentation

This site contains internal documentation, standards, and runbooks maintained by the Infrastructure team.

> This is a living document. All content should be updated via pull request.

---

## What you’ll find here

- 📋 **Standards** – Baseline configurations and enterprise guidelines  
- 🔧 **Runbooks** – Step-by-step operational procedures  
- 🚀 **Build & Automation** – VM builds, templates, and pipelines  
- 🔐 **Security & Compliance** – CIS controls and hardening notes  

---

## How to contribute

1. Edit or add Markdown files under the `docs/` directory  
2. Submit a pull request  
3. Have another team member review and approve
